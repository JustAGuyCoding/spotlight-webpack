Spot On!

17 Sept 2020

Spotlight is a HTML blog template, released under the Apache 2.0 License.

It uses Montserrat and Raleway fonts for an easily readable, light and modern, feel. The colour palette is simply dark-grey and white, with a little blue thrown in for links.

The site-name "Spotlight" is displayed in the top-left; to the right is the navigation bar which collapses to a hamburger menu for smaller devices. A search input box is provided to the top right.

The main feature is provided using a Jumbotron, displaying a banner image, accompanying Call to Action text and a link to the main content; two minor features are tucked underneath using Card Posts, showing small images and text for secondary Call to Action items

Being Bootstrap based, the template is fully responsive and will resize to fit smaller screens automatically.

A list of links pointing to older posts and labels is provided on the right hand-side, with space for other widgets and gadgets to be added below.

Blog traversal is performed using buttons at the bottom of the page, where there is also a footer providing space for additional link lists, branding and notifications.

Lighten Up!

10 Aug

The HTML template comes packaged in two different forms:

  1. As a basic HTML Template
  2. As a Webpack Starter Kit

The advantage of the Webpack Starter Kit is that it allows you to jump straight into a pre-grown development environment and customise the template. It supports inline CSS, CSS and JavaScript minification and automated cache busting.

The Starter Kit may also be used to help reduce the template's default Bootstrap CSS and JavaScript dependencies.
